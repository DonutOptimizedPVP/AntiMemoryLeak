# AntiMemoryLeak (Minecraft 1.21.x)
An advanced, lightweight memory optimization mod for Fabric, designed to replace old memory leak fixes.

## 🚀 Key Features
- **Smart GC:** Triggered cleanup based on memory thresholds.
- **Leak Fixes:** Specifically targets Chunks, Shaders, and GUI screens.
- **Zero Configuration:** Works out of the box to stabilize FPS.

## 🛠 Tech Stack
- **Language:** Java 21
- **Platform:** Fabric Loader
- **Package:** `pvp.vnnhattruongneee.antimemoryleak`
